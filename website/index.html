About App

Cleaner Royall is an open-source Android cleaner app built for speed, precision, and user privacy. No unnecessary permissions, no trackers. Just power.

Features

  • Smart Cleaner with rules
  • Magisk & Xposed module support
  • WhatsApp/Telegram data cleaner
  • Auto-clean scheduling
  • Trash & cache cleaner
  • No permissions needed

Usage

Simply install the app and open it. No permissions needed. Root required for advanced tools like fstrim, log wiping, and custom module control.
